# Bignum
使用c++实现大整数的运算

---
## 重载运算符
**已实现：**
- [x] + `大数相加，a + b` 
- [x] - `大数相减，a - b`
- [x] * `大数相乘，a * b`
- [x] / `大数相除，a / b`
- [x] > `大于`
- [x] < `小于`
- [x] == `等于`
- [x] >> `输入流，支持链式赋值`
- [x] << `输出流，支持链式调用`

**未实现：**
- [ ] - `负运算，-a`
- [ ] ++ `自增，a++ & ++a`
- [ ] -- `自减，a-- & --a`
- [ ] += `a += b`
- [ ] -= `a -= b`
- [ ] *= `a *= b`
- [ ] /= `a /= b`
- [ ] >= `大于等于，a >= b`
- [ ] <= `小于等于，a <= b`
